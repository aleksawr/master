# Simulation Study: Measurement Error and Structural Complexity in Prediction

This simulation study investigates how **measurement error** and **signal structure** jointly influence predictive performance in **regression models** and **machine learning algorithms**.

The design extends classical regression logic by explicitly modeling:

- latent (true) predictors and outcomes  
- controlled signal composition (linear vs. interaction effects)  
- measurement error in both predictors and outcome  

The goal is to evaluate whether flexible models retain advantages when data quality is limited.

---

## Data-Generating Process

The simulation separates **latent (true)** variables from **observed (noisy)** variables, following classical measurement error theory.

---

### 1. Latent Predictors

Latent predictors are generated as independent standard normal variables:

$$
X_j^* \sim \mathcal{N}(0, 1), \quad j = 1, \dots, p
$$

```r
X_true <- matrix(rnorm(N * p, 0, 1), nrow = N, ncol = p)
```

These represent **error-free variables**.

---

### 2. Latent Signal

The outcome is generated from a latent signal \( \eta \), combining linear and interaction effects.

#### Linear component

$$
\eta_{\text{linear}} =
0.6X_1^* + 0.5X_2^* + 0.4X_3^* + 0.3X_4^*
$$

```r
linear_signal_raw <- as.vector(X_true %*% beta)
```

#### Interaction component

$$
\eta_{\text{interaction}} = X_1^* \cdot X_2^*
$$

```r
interaction_signal_raw <- X_true[, "X1"] * X_true[, "X2"]
```

#### Standardization

Both components are standardized:

```r
linear_signal_z <- as.numeric(scale(linear_signal_raw))
interaction_signal_z <- as.numeric(scale(interaction_signal_raw))
```

#### Signal composition

$$
\eta = w_{\text{linear}} \cdot \tilde{\eta}_{\text{linear}} +
      w_{\text{interaction}} \cdot \tilde{\eta}_{\text{interaction}}
$$

$$
w_{\text{linear}} = \sqrt{c_{\text{linear}}}, \quad
w_{\text{interaction}} = \sqrt{1 - c_{\text{linear}}}
$$

```r
w_linear <- sqrt(comp_linear)
w_interaction <- sqrt(1 - comp_linear)

signal <- w_linear * linear_signal_z +
          w_interaction * interaction_signal_z
```

---

### 3. Latent Outcome

$$
Y^* = \eta + \epsilon
$$

Residual variance is calibrated to achieve target \( R^2 \):

$$
\text{Var}(\epsilon) =
\text{Var}(\eta) \cdot \frac{1 - R^2}{R^2}
$$

```r
var_signal <- var(signal)
var_error <- var_signal * (1 - latent_R2) / latent_R2

eps <- rnorm(N, 0, sqrt(var_error))
Y_true <- signal + eps
```

---

### 4. Measurement Error

Observed variables are generated by adding classical measurement error.

$$
X = X^* + u_X, \quad Y = Y^* + u_Y
$$

Error variance:

$$
\sigma^2_{\text{error}} =
\sigma^2_{\text{true}} \cdot \frac{1 - \rho}{\rho}
$$

```r
add_measurement_error <- function(true_var, rho) {
  var_true <- var(true_var)
  var_error <- var_true * (1 - rho) / rho
  error <- rnorm(length(true_var), 0, sqrt(var_error))
  true_var + error
}
```

```r
X_obs <- apply(X_true, 2, add_measurement_error, rho = rho_X)
Y_obs <- add_measurement_error(Y_true, rho_Y)
```

---

## Simulation Conditions

- Latent \( R^2 \): \( \{0.20, 0.50, 0.80\} \)  
- Predictor reliability \( \rho_X \): \( \{0.60, 0.80, 1.00\} \)  
- Outcome reliability \( \rho_Y \): \( \{0.60, 0.80, 1.00\} \)  
- Signal composition \( c_{\text{linear}} \): \( \{0.20, 0.50, 0.80\} \)

## Models

### Baseline OLS

$$
Y = \beta_0 + \sum_{j=1}^{p} \beta_j X_j + \varepsilon
$$

### Correctly Specified OLS

$$
Y = \beta_0 + \sum_{j=1}^{p} \beta_j X_j + \gamma (X_1 X_2) + \varepsilon
$$

### Machine Learning (XGBoost)

$$
Y = f(X) + \varepsilon
$$

---

## Evaluation Metrics

### RMSE

$$
RMSE =
\sqrt{\frac{1}{n} \sum (Y_i - \hat{Y}_i)^2}
$$

### R²

$$
R^2 =
1 - \frac{\sum (Y_i - \hat{Y}_i)^2}{\sum (Y_i - \bar{Y})^2}
$$

---

## Key Theoretical Constraints

Predictive performance is constrained by:

- signal strength  
- measurement quality  

### Measurement ceiling

$$
R^2_{\text{observed}} \leq \rho_Y
$$
